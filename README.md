# Insurance-Risk-Claim-Analysis-Reports
Car Insurance Risk & Claims Analysis using Data Analytics | Power BI Dashboard | Customer Segmentation & Risk Profiling
This project analyzes car insurance data to identify risk patterns, customer behavior, and claim trends.

Objectives
- Identify high-risk customers
- Analyze claim frequency & amount
- Segment customers based on demographics
- Improve insurance pricing strategies.

Dataset Details
- Customer info (Age, Gender, Income)
- Car details (Model, Year, Usage)
- Claims (Amount, Frequency)

Key Insights
- Young drivers → higher risk
- Commercial cars → more claims
- Multiple kids driving → higher accident chances

This project helps insurance companies understand risk better and optimize pricing.
